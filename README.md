## Spring 1기 최종 과제: 동물 관리 시스템
#### 1. 프로젝트 설명
* **콘솔 기반**의 동물 관리 시스템
* 동물원에서 여러종의 동물들을 등록및 관리 한다
* 동물은 `이름`, `나이`, `포만도`, `행복도` 를 가진다 
* `먹이주기`, `놀아주기` 시스템이 있다
* 각 동물들은 **고유의 울음**을 가진다
